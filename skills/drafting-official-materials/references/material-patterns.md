# 机关材料写作模式

Use this file for non-statutory but formal Party/government materials.

## 讲话稿

Core sequence:

1. Recognize the situation: why this meeting/topic matters.
2. Affirm achievements or identify gaps.
3. Deploy key tasks: usually 3-5 major points.
4. Strengthen guarantees: leadership, responsibility, coordination, discipline, supervision.
5. Mobilize action: concise closing.

Tone:

- More forceful than a report, less rigid than a statutory document.
- Use audience-aware phrasing: `同志们`, `各单位要`, `要切实增强`.

## 汇报材料

Core sequence:

1. Basic situation.
2. Main practices and results.
3. Existing problems.
4. Next work plan or requests for support.

Writing emphasis:

- Use facts and measures before conclusions.
- Put distinctive practices in parallel headings.

## 工作总结

Core sequence:

1. Overall assessment.
2. Main work completed.
3. Results and highlights.
4. Problems and不足.
5. Next-stage plan.

Avoid:

- Only listing activities without outcomes.
- Treating problems as empty self-criticism.

## 调研报告

Core sequence:

1. Research background and method.
2. Current situation and findings.
3. Main problems.
4. Cause analysis.
5. Countermeasures and recommendations.

Quality bar:

- Findings should be specific, not generic.
- Recommendations should match the authority and resources of the target reader.

## 信息简报 / 通讯稿

Core sequence:

1. Lead: one sentence for the most important news.
2. Facts: time, place, subject, action.
3. Measures: what was done.
4. Effect: what changed or what value emerged.
5. Next step: optional.

Tone:

- More readable than a formal document, still institutional.
- Do not exaggerate or write advertising language.
